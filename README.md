# AI-Poetry
AI Poetry - ქართული პოეზიის გენერატორი ხელოვნური ინტელექტის დახმარებით. შექმენით ექსპერიმენტული ლექსები სხვადასხვა სტილით და განწყობით. 
## 📖 აღწერა

**AI Poetry** არის ინტერაქტიული ვებ აპლიკაცია, რომელიც აგენერირებს ექსპერიმენტულ პოეზიას Claude AI-ის გამოყენებით. აპლიკაცია საშუალებას გაძლევთ შექმნათ AI პოეზია სხვადასხვა სტილით, განწყობით, საკუთარი მეტაფორებით.

## ✨ ფუნქციები

- 🎨 **4 პოეტური სტილი:** ვერლიბრი, თავისუფალი, მოდერნისტული, სიმბოლისტური
- 🎭 **6 განწყობა:** ნეიტრალური, მელანქოლიური, იუმორისტული, ოპტიმისტური, ფილოსოფიური, რომანტიული
- 📏 **3 სიგრძის ვარიანტი:** მოკლე, საშუალო, გრძელი
- 🔮 **მეტაფორების კონტროლი:** საკუთარი მეტაფორების დამატება ან AI-ის თავისუფალი გენერაცია
- 💾 **შენახვის ფუნქცია:** გენერირებული ლექსების შენახვა და მართვა
- 📋 **კოპირება:** ერთი დაჭერით ლექსის კოპირება

3. დააყენეთ სასურველი ზომა
4. სურვილის შემთხვევაში ჩართეთ მეტაფორები და დაამატეთ საკუთარი მეტაფორა
